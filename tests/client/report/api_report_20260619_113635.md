## Phân Tích Nguyên Nhân

### 🔴 Critical — Ảnh hưởng chức năng chính

| # | Vấn đề | Endpoint | Root Cause |
|---|---|---|---|
| 1 | **CVE Search 500** | `POST /api/v2/cves/search` | Elasticsearch index rỗng — chưa sync data. Kiểm tra: `STARTUP_SYNC_ENABLED=true` nhưng NVD_API_KEY trống → rate limit fail |
| 2 | **Scans 404** | `GET /api/v1/scans` | Gateway chưa register route `/api/v1/scans` |
| 3 | **Findings 500** | `GET /api/v1/findings` | DB query lỗi — có thể `scan_id` FK constraint hoặc table rỗng |
| 4 | **SLA Overview 404** | `GET /api/v1/sla/overview` | Route chưa implement trong gateway |
| 5 | **AI Service 503** | `GET /ai/triage/queue` | AI enrichment service chưa chạy (OPENAI_API_KEY trống) |

### 🟠 Schema Mismatch — Data đúng nhưng format sai

| # | Endpoint | Spec | Thực tế |
|---|---|---|---|
| 1 | `GET /auth/me` | `{ "user": {...} }` | User object trực tiếp |
| 2 | `POST /auth/logout` | 204 No Content | 200 OK |
| 3 | `GET /kev` | `{ "data": [...], "page_size": N, "stats": {...} }` | `{ "items": [...], "count": N }` |
| 4 | `GET /assets` | `{ "assets": [...], "total": N }` | Object trực tiếp |
| 5 | `GET /products` | `{ "products": [...], "total": N }` | List trực tiếp |
| 6 | `GET /admin/health` | `{ "services": { "redis": {...} } }` | `{ "services": [{...}] }` |
| 7 | `GET /admin/users` | User có `name`, `mfa_enabled` | Thiếu `name`, `mfa_enabled` |
| 8 | `GET /admin/roles` | `{ "name": "...", "permissions": [...] }` | Key khác (`role_name`, `perms`) |
| 9 | `GET /webhooks` | Array trực tiếp | `{ "webhooks": [...] }` |
| 10 | `GET /products/types` | `["string", ...]` | `[{ "id": ..., "name": ... }]` |
